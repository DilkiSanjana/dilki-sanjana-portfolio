# Dilki Sanjana - Personal Portfolio

A modern, responsive, and interactive personal portfolio showcasing my projects, skills, education, and professional background. Built with **React** and **Vite**, this portfolio is designed with a sleek glassmorphism UI, smooth animations, and a dynamic dark/light mode experience.

## ✨ Features

- **Modern Tech Stack:** Built with React 19 and Vite for fast development and optimized production builds.
- **Dynamic Theming:** Seamless Dark and Light mode support with a centralized CSS variable system.
- **Interactive UI:** Smooth scroll, fade-in animations (via AOS), and a typing hero section (via typed.js).
- **Glassmorphism Design:** Beautiful translucent cards and modern gradients.
- **Live GitHub Contributions:** Real-time fetching of GitHub activity using `react-github-calendar` and third-party APIs.
- **Responsive Layout:** fully optimized for mobile, tablet, and desktop views.
- **Interactive CV Modal:** Easily view and access my latest CV without leaving the page.

## 🚀 Technologies Used

- **Framework:** React 19
- **Build Tool:** Vite
- **Styling:** Custom CSS with Glassmorphism and CSS Variables
- **Animations:** AOS (Animate On Scroll), typed.js, custom CSS keyframes
- **GitHub Integration:** `react-github-calendar`
- **Icons:** Bootstrap Icons, FontAwesome
